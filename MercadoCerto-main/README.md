MercadoCerto

Descrição do Projeto

O MercadoCerto é um sistema desenvolvido para facilitar o cadastro, consulta e avaliação de mercados, produtos e usuários.
O projeto inclui API REST em Spring Boot, além de páginas estáticas para interação rápida com o sistema.

Ele foi criado para simular o funcionamento de uma startup: comparar mercados, exibir rankings, registrar produtos e permitir que usuários avaliem estabelecimentos.
